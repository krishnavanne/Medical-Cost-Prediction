# 🏥 Medical Cost Prediction using Machine Learning

## 📌 Overview
This project aims to predict medical insurance costs based on user attributes such as age, BMI, smoking habits, and region. Machine Learning regression models are used to estimate the charges accurately.

---

## 🎯 Problem Statement
Medical costs vary significantly based on multiple factors. The goal of this project is to build a predictive model that can estimate insurance charges for individuals.

---

## 📊 Dataset Information
The dataset contains the following features:

- **age**: Age of the person  
- **sex**: Gender  
- **bmi**: Body Mass Index  
- **children**: Number of children/dependents  
- **smoker**: Smoking status  
- **region**: Residential area  
- **charges**: Medical cost (Target variable)

---

## ⚙️ Tech Stack
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Scikit-learn  
- Streamlit  

---

## 🔍 Exploratory Data Analysis (EDA)
- Checked missing values  
- Data visualization using graphs  
- Feature relationships analyzed  
- Outliers detection  

---

## 🧠 Machine Learning Models Used
- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  

---

## 📈 Model Performance
- Evaluated using:
  - R² Score  
  - Mean Absolute Error (MAE)  
  - Mean Squared Error (MSE)
    
Machine Learning project to predict medical costs using regression models and deployed with Streamlit.
