# Medical-Cost-Prediction
# 💊 Medical Cost Prediction using Machine Learning

## 📌 Project Overview

This project predicts medical expenses based on features like age, BMI, smoking habits, and number of children using multiple machine learning regression models.

---

## 🚀 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature encoding (categorical to numerical)
* Multiple regression models:

  * Linear Regression
  * Polynomial Regression
  * Ridge Regression
  * Lasso Regression
  * ElasticNet
* Model evaluation using:

  * MAE (Mean Absolute Error)
  * MSE (Mean Squared Error)
  * R2 Score
* Best model selection
* Streamlit web app for real-time predictions

---

## 🛠️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* Streamlit

---

## 📊 Exploratory Data Analysis

EDA was performed to understand relationships between features and medical costs.
Key insights:

* Smoking significantly increases medical costs
* Age and BMI are strongly correlated with expenses

---

## 🤖 Machine Learning Models

Multiple regression models were implemented and compared:

| Model                 | Description                                        |
| --------------------- | -------------------------------------------------- |
| Linear Regression     | Basic regression model                             |
| Polynomial Regression | Captures non-linear relationships                  |
| Ridge Regression      | Reduces overfitting using L2 regularization        |
| Lasso Regression      | Performs feature selection using L1 regularization |
| ElasticNet            | Combination of Ridge and Lasso                     |

---

## 📈 Model Evaluation

Models were evaluated using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R2 Score
👉 Polynomial Regression achieved the best performance.

## 🎯 Conclusion
Polynomial Regression performed best by capturing non-linear relationships in the data.
Regularization techniques like Ridge, Lasso, and ElasticNet helped reduce overfitting and improve generalization.

Machine Learning project to predict medical costs using regression models and deployed with Streamlit.
